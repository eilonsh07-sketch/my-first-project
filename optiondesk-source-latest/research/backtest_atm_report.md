## 10. מסקנה ופסיקה

### ATM לעומת OTM — מה עדיף?

**עבור המשקיע:**

| קריטריון | עדיף ATM? |
|---|---|
| Hit-rate (שיעור עסקאות רווחיות) | ✅ כן — +6-8 נ"ב גבוה יותר |
| Median return | ✅ כן — הפסד חציוני קטן יותר ב-12mo (−43.7% לעומת −100%) |
| "הגנה" מפני אפיסה מלאה | ✅ כן — פחות −100% outcomes |
| Mean return ב-12mo | ❌ לא — 118.4% לעומת 129.8% |
| "פוטנציאל גבוה" (best case) | ❌ לא — max ATM 12,240% לעומת OTM 13,537% |
| כוח ניבוי (Spearman 6mo) | = שווה (0.120 לעומת 0.123) |

### פסיקה:

> **ATM קולס הן אפשרות סבירה ושמרנית יותר מ-OTM, אך עדיין ספקולטיביות ביסוד.**
>
> **יתרון ATM:** hit-rate גבוה יותר בכ-7 נקודות בסיס, חציון הרבה פחות גרוע ב-12mo (−44% לעומת −100%), ופחות עסקאות שמסתיימות בהפסד מלא.
>
> **חיסרון ATM:** mean return ב-12mo נמוך מעט (118% לעומת 130%), ופחות leverage בעסקאות הניצחות הגדולות.
>
> **מה שלא השתנה:** החציון נשאר שלילי בכל האופקים. רוב העסקאות מסתיימות בהפסד (>57% מהזמן). הסקנר מציג כוח ניבוי חיובי רק ב-6mo (מובהק סטטיסטית), לא ב-12mo. **אי אפשר להסתמך על החציון כתמורה ריאלית — הממוצעים מונעים ע"י outliers קיצוניים (LITE +3,186%).**

### שורה תחתונה — ATM vs OTM:
```
ATM:  12mo hit=42.8%, mean=+118%, median=−44%, Spearman(6mo)=0.120
OTM:  12mo hit=36.0%, mean=+130%, median=−100%, Spearman(6mo)=0.123
```
**ATM עדיף למי שרוצה פחות סיכון של אפיסה מוחלטת; OTM עדיף למי שמחפש leverage מקסימלי על תנועות חזקות.**
